# Earth Tone Tarot Reading

A responsive three-card tarot reading web app built with Next.js, React, TypeScript, Tailwind CSS, and Framer Motion.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## GitHub Pages

This project is configured for GitHub Pages at:

```text
https://tonfernz.github.io/tarottest/
```

Pushes to `main` trigger `.github/workflows/deploy.yml`, which exports the Next.js app as static files.
